目的和背景
---
* **希望通过本项目帮助8-16岁的孩子了解并在生活中使用信息学。

作为两个8岁孩子的父亲，我深深的被他们的知识量所震惊。

当我问他们我们看到的世界和蛇看到的世界不一样，那世界是真实的吗？
####他们回答我是真实的。
又问他们，那栋楼的颜色就是我们看到的吗？
他们告诉我不是，人只能看到3种颜色，鸽子可以看到4种。
又问他们，鸽子为什么能看到4种呢？
他们告诉我，人只有3种色细胞，鸽子有4种色细胞，所以。
我震惊了。

我能做什么？
